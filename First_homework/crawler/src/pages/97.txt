Я же хочу более детально остановиться на самом алгоритме, заглянуть в его реализацию и продемонстрировать его работу на примере моей программы.<br/>
<br/>
Итак, в чем же заключается суть методики Adaboost? <br/>
<a name="habracut"></a><br/>
Суть заключается в том, что если у нас есть набор эталонных объектов (точки на плоскости), т.е. есть значения и класс, к которому они принадлежат (например, -1 – красная точка, +1 – синяя точка), кроме того имеется множество простых классификаторов (набор вертикальных или горизонтальных прямых, которые разделяют плоскость на две части с наименьшей ошибкой), то мы можем составить один лучший классификатор. При этом в процессе составления или обучения финального классификатора акцент делается на эталоны, которые распознаются «хуже», в этом и заключается адаптивность алгоритма, в процессе обучения он подстраивается под наиболее «сложные» объекты. <br/>
<br/>
Псевдокод алгоритма хорошо описан на <a href="http://ru.wikipedia.org/wiki/AdaBoost">Вики</a>. Приведем его здесь.<br/>

<h4>Простой классификатор</h4><br/>
<br/>
Итак, простой классификатор. Как я уже говорил, это вертикальная или горизонтальная прямая, которая разделяет плоскость на две части, минимизируя ошибку классификации: <img src="https://habrastorage.org/r/w1560/getpro/habr/post_images/b26/603/a35/b26603a35859fc6ed6ccdb38f2f12eb1.png" alt="image" data-src="https://habrastorage.org/getpro/habr/post_images/b26/603/a35/b26603a35859fc6ed6ccdb38f2f12eb1.png"/> (квадратные скобки от булевого выражения равны 1, ели true, и 0, если false). На языке С# процедуру такого обучения можно написать следующим образом:<br/>
<br/>
//Обучение простого классификатора<br/>
<br/>
<code><font color="black"><font color="#0000FF">public</font> <font color="#0000FF">override</font> <font color="#0000FF">void</font> Train(LabeledDataSet&lt;<font color="#0000FF">double</font>> trainingSet)<br/>
  {<br/>
   <font color="#008000">// Инициализируем наши коэффициенты D_i</font><br/>
   <font color="#0000FF">if</font> (_weights == <font color="#0000FF">null</font>)<br/>
   {<br/>
    _weights = <font color="#0000FF">new</font> <font color="#0000FF">double</font>[trainingSet.Count];<br/>
    <font color="#0000FF">for</font> (<font color="#0000FF">int</font> i = 0; i &lt; trainingSet.Count; i++)<br/>
     _weights[i] = 1.0 / trainingSet.Count;<br/>
   }<br/>
 <br/>
   Random rnd = <font color="#0000FF">new</font> Random();<br/>
          <font color="#0000FF">double</font> minimumError = <font color="#0000FF">double</font>.PositiveInfinity;<br/>
   _d = -1;<br/>
   _threshold = <font color="#0000FF">double</font>.NaN;<br/>
   _sign = 0;<br/>
 <br/>
   <font color="#008000">// Цикл по всем координатам пространства </font><br/>
   <font color="#0000FF">for</font> (<font color="#0000FF">int</font> d = 0; d &lt; trainingSet.Dimensionality; d++)<br/>
   {<br/>
    <font color="#008000">// Для оптимизации в случае, если пространство огромной размерности</font><br/>
    <font color="#0000FF">if</font> (rnd.NextDouble() &lt; _randomize)<br/>
     <font color="#0000FF">continue</font>;<br/>
 <br/>
    <font color="#008000">// Сортируем данные для эффективного поиска прямой x_d = threshold </font><br/>
    <font color="#0000FF">double</font>[] data = <font color="#0000FF">new</font> <font color="#0000FF">double</font>[trainingSet.Count];<br/>
    <font color="#0000FF">int</font>[] indices = <font color="#0000FF">new</font> <font color="#0000FF">int</font>[trainingSet.Count];<br/>
    <font color="#0000FF">for</font> (<font color="#0000FF">int</font> i = 0; i &lt; trainingSet.Count; i++)<br/>
    {<br/>
     data[i] = trainingSet.Data[i][d];<br/>
     indices[i] = i;<br/>
    } <br/>
    Array.Sort(data, indices);<br/>
 <br/>
    <font color="#008000">// Определяем максимальную ошибку</font><br/>
    <font color="#0000FF">double</font> totalError = 0.0;<br/>
    <font color="#0000FF">for</font> (<font color="#0000FF">int</font> i = 0; i &lt; trainingSet.Count; i++)<br/>
     totalError += _weights[i];<br/>
 <br/>
    <font color="#008000">// Инициализируем значение ошибки для поиска минимальной ошибки</font><br/>
    <font color="#0000FF">double</font> currentError = 0.0;<br/>
    <font color="#0000FF">for</font> (<font color="#0000FF">int</font> i = 0; i &lt; trainingSet.Count; i++)<br/>
     currentError += trainingSet.Labels[i] == -1 ? _weights[i] : 0.0;<br/>
 <br/>
    <font color="#008000">// Ищем в отсортированном массиве лучший параметр threshold, который будет минимизировать </font><br/>
                               <font color="#008000">//ошибку, и определяем знак классификатора</font><br/>
    <font color="#0000FF">for</font> (<font color="#0000FF">int</font> i = 0; i &lt; trainingSet.Count - 1; i++)<br/>
    {<br/>
     <font color="#008000">// Обновляем ошибку при рассмотрении текущего объекта</font><br/>
     <font color="#0000FF">int</font> index = indices[i];<br/>
     <font color="#0000FF">if</font> (trainingSet.Labels[index] == +1)<br/>
      currentError += _weights[index];<br/>
     <font color="#0000FF">else</font><br/>
      currentError -= _weights[index];<br/>
 <br/>
     <font color="#008000">// Если идут одинаковые данные, то порог не ищем</font><br/>
     <font color="#0000FF">if</font> (data[i] == data[i + 1])<br/>
      <font color="#0000FF">continue</font>;<br/>
 <br/>
     <font color="#008000">// Определяем потенциально возможное значение для threshold </font><br/>
     <font color="#0000FF">double</font> testThreshold = (data[i] + data[i + 1]) / 2.0;<br/>
 <br/>
     <font color="#008000">// Запоминаем значение threshold, если ошибка минимальна, и определяем знак</font><br/>
     <font color="#0000FF">if</font> (currentError &lt; minimumError) <font color="#008000">// Классификатор с _sign = +1</font><br/>
     {<br/>
      minimumError = currentError;<br/>
      _d = d;<br/>
      _threshold = testThreshold;<br/>
      _sign = +1;<br/>
     }<br/>
     <font color="#0000FF">if</font> ((totalError - currentError) &lt; minimumError) <font color="#008000">// Классификатор с _sign = -1</font><br/>
     {<br/>
      minimumError = (totalError - currentError);<br/>
      _d = d;<br/>
      _threshold = testThreshold;<br/>
      _sign = -1;<br/>
     }<br/>
    }<br/>
   }<br/>
  }</font></code><br/>
<br/>
Если Вы заметили, то процедура обучения простого классификатора написана таким образом, что если ему передать веса, которые получаются на протяжении работы алгоритма AdaBoost, то тогда будет искаться тот классификатор, который минимизирует взвешенную ошибку классификации п.1. Тогда сам алгоритм на С# будет выглядеть следующим образом:<br/>
<br/>
<h4>«Сложный» классификатор</h4><br/>
<br/>
//Обучение с помощью AdaBoost<br/>
<br/>
<code><font color="black"><font color="#0000FF">public</font> <font color="#0000FF">override</font> <font color="#0000FF">void</font> Train(LabeledDataSet&lt;<font color="#0000FF">double</font>> trainingSet)<br/>
  {<br/>
   <font color="#008000">// Инициализируем веса D_i</font><br/>
   <font color="#0000FF">if</font> (_weights == <font color="#0000FF">null</font>)<br/>
   {<br/>
    _weights = <font color="#0000FF">new</font> <font color="#0000FF">double</font>[trainingSet.Count];<br/>
    <font color="#0000FF">for</font> (<font color="#0000FF">int</font> i = 0; i &lt; trainingSet.Count; i++)<br/>
     _weights[i] = 1.0 / trainingSet.Count;<br/>
   }<br/>
 <br/>
   <font color="#008000">// Делаем непосредственно обучение простых классификаторов</font><br/>
   <font color="#0000FF">for</font> (<font color="#0000FF">int</font> t = _h.Count; t &lt; _numberOfRounds; t++)<br/>
   {<br/>
    <font color="#008000">// Создаем простой классификатор и обучаем его</font><br/>
    WeakLearner h = _factory();<br/>
    h.Weights = _weights;<br/>
    h.Train(trainingSet);<br/>
 <br/>
    <font color="#008000">//Вычисляем ввзвешенную ошибку классификации </font><br/>
    <font color="#0000FF">int</font>[] hClassification = <font color="#0000FF">new</font> <font color="#0000FF">int</font>[trainingSet.Count];<br/>
    <font color="#0000FF">double</font> epsilon = 0.0;<br/>
    <font color="#0000FF">for</font> (<font color="#0000FF">int</font> i = 0; i &lt; trainingSet.Count; i++)<br/>
    {<br/>
     hClassification[i] = h.Classify(trainingSet.Data[i]);<br/>
     epsilon += hClassification[i] != trainingSet.Labels[i] ? _weights[i] : 0.0;<br/>
    }<br/>
 <br/>
    <font color="#008000">// Если величина epsilon >= 0.5, то останавливаемся</font><br/>
    <font color="#0000FF">if</font> (epsilon >= 0.5)<br/>
     <font color="#0000FF">break</font>;<br/>
 <br/>
    <font color="#008000">// Выбираем \alpha_{t}</font><br/>
    <font color="#0000FF">double</font> alpha = 0.5 * Math.Log((1 - epsilon) / epsilon);<br/>
 <br/>
    <font color="#008000">// Обновляем коэффициенты D_i</font><br/>
    <font color="#0000FF">double</font> weightsSum = 0.0;<br/>
    <font color="#0000FF">for</font> (<font color="#0000FF">int</font> i = 0; i &lt; trainingSet.Count; i++)<br/>
    {<br/>
     _weights[i] *= Math.Exp(-alpha * trainingSet.Labels[i] * hClassification[i]);<br/>
     weightsSum += _weights[i];<br/>
    }<br/>
    <font color="#008000">// Нормируем полученные коэффициенты </font><br/>
    <font color="#0000FF">for</font> (<font color="#0000FF">int</font> i = 0; i &lt; trainingSet.Count; i++)<br/>
     _weights[i] /= weightsSum;<br/>
 <br/>
    <font color="#008000">// Запоминаем классификатор с найденным коэффициентом</font><br/>
    _h.Add(h);<br/>
    _alpha.Add(alpha);<br/>
 <br/>
    <font color="#008000">// Останавливаем обучение, если ошибка равна нулю</font><br/>
    <font color="#0000FF">if</font> (epsilon == 0.0)<br/>
     <font color="#0000FF">break</font>;<br/>
   }<br/>
  }</font></code><br/>
<br/>
Таким образом, после выбора оптимального классификатора h_{t} \, для распределения D_i, объекты x_{i} \,, которые классификатор h_{t} \, идентифицирует корректно, имеют веса меньшие, чем те, которые идентифицируются некорректно. Следовательно, когда алгоритм тестирует классификаторы на распределении D_{t+1}, он будет выбирать классификатор, который лучше идентифицирует объекты, неверно распознаваемые предыдущим классификатором.<br/>
<br/>
<h4>Пример</h4><br/>
<br/>
В случае классификации таких данных:<br/>
<img src="http://lh4.ggpht.com/_Tr3fxRuFwow/S0SBUNYXTRI/AAAAAAAAASg/ir2-oYjTOq4/s512/in_1.png" alt="image" data-src="http://lh4.ggpht.com/_Tr3fxRuFwow/S0SBUNYXTRI/AAAAAAAAASg/ir2-oYjTOq4/s512/in_1.png"/><br/>
<br/>
классифицированый рисунок с помощю AdaBoost будет выглядеть так:<br/>
<img src="http://lh5.ggpht.com/_Tr3fxRuFwow/S0SBUOv4h0I/AAAAAAAAASk/0Ah7GQgM5cU/s512/out_adaboost_1.png" alt="image" data-src="http://lh5.ggpht.com/_Tr3fxRuFwow/S0SBUOv4h0I/AAAAAAAAASk/0Ah7GQgM5cU/s512/out_adaboost_1.png"/><br/>
<br/>
Для сравнения можно привести простой алгоритм, который для каждой точки определяет ее ближайшего соседа, класс которого известен. (Понятно, что затраты такого алгоритма очень велики.)<br/>
<img src="http://lh6.ggpht.com/_Tr3fxRuFwow/S0SDr-iKuaI/AAAAAAAAASw/H6fAQ8fRYQY/s512/out_nn.png" alt="image" data-src="http://lh6.ggpht.com/_Tr3fxRuFwow/S0SDr-iKuaI/AAAAAAAAASw/H6fAQ8fRYQY/s512/out_nn.png"/><br/>
<br/>
<h4>Заключение</h4><br/>
<br/>
Как видим, алгоритм, построенный с помощью AdaBoost, классифицирует намного лучше, чем один классификатор, но и он имеет ошибки. Но простота и изящность алгоритма позволила ему войти в <a href="http://www.cs.umd.edu/~samir/498/10Algorithms-08.pdf">Top 10 algorithms in data mining</a>. Надеюсь, код, приведенный в статье, помог вам разобраться с этим алгоритмом.<br/>
<br/>
